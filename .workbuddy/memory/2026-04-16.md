# 2026-04-16 开发日志

## 完成内容

### 物品系统基础
- **ItemSO.cs**：ScriptableObject 基类，创建 `Assets/Scripts/Items/ItemSO.cs`
  - 字段：itemName, itemDescription, icon, isStackable, maxStack
  - 右键 → Create → Item → New Item 创建物品数据文件

### 快捷栏UI
- **InventorySlot.cs**：单个格子组件
  - 显示物品图标（Image）
  - 显示堆叠数量（Text，隐藏状态）
  - UpdateUI() 更新显示
  - SetItem() 设置物品和数量
- **InventoryManger.cs**：单例管理器
  - 管理7个格子数组
  - AddItem() 核心逻辑：可堆叠优先叠加，否则找空格子
  - 背包满时 Debug.Log 提示
- **Canvas结构**：BottomBar → InventoryPanel → 7个 InventorySlot

### 物品收集
- **Loot.cs**：物品掉落脚本
  - OnTriggerEnter2D 检测 Player
  - 调用 InventoryManger.instance.AddItem()
  - 播放拾取动画后销毁
- **Prefab制作**：Meat.prefab 挂载 Sprite Renderer + Collider2D + Animator + Loot脚本

### 堆叠功能
- ItemSO.isStackable 字段控制
- 优先叠加同类物品到 maxStack 上限

### 资源创建
- Meat Resource (肉类)：isStackable=true, maxStack=99
- Wood Resource (木头)：isStackable=true, maxStack=99

## 遇到的问题
1. **Animator Controller Invalid**：动画状态名写错，重新创建Controller解决
2. **"背包已满"误报**：Slots数组的Icon Image和Count Text未在Inspector中关联
3. **数量文字不显示**：countText.enabled在物体未激活时无效，改用 SetActive
