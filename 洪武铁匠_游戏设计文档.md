# 《洪武铁匠》游戏设计文档

> 一个关于在乱世中用锤子活下来的故事

---

## 一、游戏定位

**类型**：2D卡牌RPG + 模拟经营

**核心主题**：
> 铁匠虽然制造的是毁灭世界的刀剑，但铁匠是为和平制造刀剑。

**一句话简介**：
> 在元末明初的乱世中，你是一个铁匠。你不追求成为将军，只想用你的手艺活下来，并为秩序尽一份力。但历史不会让你置身事外——你的选择，将决定你的命运。

---

## 二、核心玩法循环

```
接任务 → 打铁/收集材料 → 完成任务
    ↑                          ↓
    ← ← ← ← ← ← ← ← ← ← ← ← ←

任务奖励：
├─ 材料（铁、木炭、其他资源）
├─ 图纸（学会制作新装备）
├─ 贡献度（阵营认可度）
└─ 剧情推进（解锁新章节）

战斗后：
├─ 胜利：获得战利品
├─ 失败：装备耐久下降/报废
└─ 撤退：保留部分装备
```

---

## 三、战斗系统

### 3.1 战斗模式

**基于卡牌的对战系统**

```
【你】                    【敌人】
铁匠                      元兵首领
锤、钳、铁条              刀、枪、火把
徒弟卡                    士兵卡
工具卡                    武器卡
                          骑兵卡

对战方式：轮流出牌
你的风格 → 工具流、防御流、守势
敌人风格 → 暴力流、进攻压制
```

### 3.2 敌人类型

| 敌人 | 风格 | 代表卡牌 |
|------|------|---------|
| 元兵 | 普通士兵 | 刀、枪 |
| 土匪 | 亡命徒 | 匕首、绊索 |
| 义军散兵 | 友军变敌人 | 矛盾（矛盾对立） |
| 将领 | 精英怪 | 战马、令旗、弓弩 |
| 大BOSS | 某历史人物 | 专属卡组 |

### 3.3 胜负条件

| 结果 | 效果 |
|------|------|
| 胜利 | 敌人生命值归零，获得战利品、图纸、经验 |
| 失败 | 你的生命值归零，被俘虏/逃跑，丢失资源 |
| 撤退成功 | 保留部分装备，任务失败但不死亡 |

### 3.4 装备与耐久度

```
【装备卡牌】

┌─────────────────────────┐
│   【精钢刀】             │
│   ⚔ 攻击+10             │
│   🛡 耐久度：100%        │
│   "火候到位，刀身笔直"   │
└─────────────────────────┘

战斗后可能变为：

┌─────────────────────────┐
│   【精钢刀·破损】        │
│   ⚔ 攻击+5              │
│   🛡 耐久度：45%         │
│   "卷刃了，需要修理"     │
└─────────────────────────┘
```

| 战斗结果 | 装备状态 |
|---------|---------|
| 胜利 | 30%概率轻微损坏 |
| 失败 | 80%概率损坏，20%概率完全报废 |
| 撤退成功 | 10%概率轻微损坏 |

**完全报废的装备只能拆解成材料**

## 四、铁匠系统（核心深度系统）

### 4.1 打铁基础

```
普通的打铁：锤子砸下去，火星飞溅
         → 做出锄头、菜刀、农具

精细的打铁：看颜色、听声音、判断时机
         → 做出好刀、铠甲、马蹄铁

大师级的打铁：知道这块铁的前世今生
            → 打出传说级武器
```

### 4.2 装备等级

| 等级 | 能打的东西 | 效果 |
|------|-----------|------|
| 学徒 | 农具、铁条 | 基础 |
| 熟练 | 刀、矛、盾 | 实用 |
| 师傅 | 铠甲、马蹄铁、攻城零件 | 精良 |
| 大师 | 传说中的好刀 | 稀有 |

**打铁不是"点击升级"，而是：**
- 你在实际打铁的过程中积累经验
- 见过越多的武器图纸，打得越好
- 收的徒弟多了，效率越高

### 4.3 资源管理

| 资源 | 来源 |
|------|------|
| 铁 | 战场废铁、商人购买、拆解装备 |
| 木炭 | 购买、自己烧制 |
| 其他材料 | 探索获得、任务奖励 |

## 五、收徒系统

### 5.1 徒弟招募

```
你在战场上/难民营/村庄里遇到：
- 孤儿
- 失去家人的年轻人
- 想学手艺的农民

你可以选择：
- 收为徒弟 → 跟你学打铁
- 给点钱打发走
- 不管
```

### 5.2 徒弟特点

| 徒弟 | 特点 | 成长方向 |
|------|------|---------|
| 小石头 | 沉默寡言，手很稳 | 擅长精细活 |
| 二狗 | 活泼好动，学得快 | 擅长创新 |
| 老张 | 以前是铁匠，被打散了 | 经验丰富，但老了 |

### 5.3 徒弟功能

```
徒弟会：
- 帮你打下手（效率提升）
- 在你没有材料时外出收集
- 在战斗中帮你背工具、搬东西
- 学成后可以独立打铁
```

## 六、阵营与贡献度系统

### 6.1 阵营选择

```
┌─────────────────────────────────────────────┐
│  【阵营A：朱元璋/郭子兴】                    │
│  贡献度权重：★★★★★                         │
│                                             │
│  - 贡献度越高，你在队伍里的地位越高          │
│  - 贡献度影响你能接的任务难度                │
│  - 贡献度影响结局                            │
│  - 历史主线：鄱阳湖之战 → 建立明朝            │
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐
│  【阵营B：陈友谅】                           │
│  贡献度权重：★★                             │
│                                             │
│  - 贡献度作用有限，历史大势不可逆            │
│  - 贡献度极高时：鄱阳湖可能有不同走向        │
│  - 但依然很难改变结局                        │
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐
│  【阵营C：张士诚】                           │
│  贡献度权重：★★                             │
│                                             │
│  - 盐商起家，财力雄厚但武力一般              │
│  - 贡献度极高：苏州可能晚破几天               │
│  - 但改变不了根本结局                        │
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐
│  【阵营D：元军】                             │
│  贡献度权重：★                              │
│                                             │
│  - 你是个铁匠，不是士兵                      │
│  - 贡献度极高：你可能被派去打造攻城器        │
│  - 但元朝注定要亡                            │
│  - 结局：要么跑，要么一起死                  │
└─────────────────────────────────────────────┘
```

### 6.2 贡献度与地位

| 贡献度 | 你能做的事 |
|--------|-----------|
| 0-10% | 打造普通装备、送货 |
| 10-30% | 带小部队打仗、参与战斗 |
| 30-50% | 带领一队工匠、负责一个战场 |
| 50-70% | 成为将领、参与战略决策 |
| 70-90% | 独当一面、影响历史进程 |
| 90%+ | 逆天改命（仅限朱元璋线有机会） |

## 七、任务系统

```
【任务面板】

主线任务：跟随某个阵营
├─ 投奔郭子兴
│   ├─ 任务1：运送粮草
│   ├─ 任务2：打造兵器

│
└─ 特殊任务：
    ├─ "收一个徒弟" → 解锁【徒弟系统】
    └─ "打一把名刀" → 解锁传说级装备
```

## 八、卡牌系统

### 8.1 卡牌来源

| 你做的事 | 获得卡牌 |
|---------|---------|
| 打铁 | 工具卡（锤子、钳子、砧板） |
| 收徒 | 徒弟卡（每个人都是一张卡） |
| 打装备 | 装备卡（刀、矛、盾） |
| 做陷阱 | 机关卡（绊索、落石） |
| 修城 | 建筑卡（城墙、营寨） |

### 8.2 卡牌升级

```
【锤子卡】Lv.1
→ 普通锤子，打人的锤子
↓ 升级条件：用来打铁100次
↓
【锤子卡】Lv.2
→ 打铁老手的锤子，更稳更准
↓ 升级条件：教出1个徒弟
↓
【锤子卡】Lv.3
→ 铁匠大师的锤子，名震一方
↓ 升级条件：为军队打过一批兵器
↓
【锤子卡】Lv.MAX
→ "洪武之锤"——传说中朱元璋用过的那把
```

## 九、剧情开篇

### 9.1 历史背景

**1352年，淮西地区**

元末乱世，赋税越来越重。各地起义军此起彼伏，元朝派兵镇压，所过之处烧杀抢掠。百姓要么逃，要么被抓壮丁，要么被杀。

### 9.2 主角设定

你是一个濠州城外的铁匠。手艺人，吃得饱饭，在村里还算有地位。

元末乱世，你只是想打铁、种地、活下去。

但这世道，不让你安宁。

### 9.3 开场：难民逃难

```
【开场：难民队伍中】

你和一群难民正在逃命。

几天前，元军围剿了一个村子。
你们侥幸逃出来，跟着人流往南走。

没有人知道目的地在哪。
只是听说"那边没有兵"。

...

忽然，后面传来马蹄声。
有人在喊："元兵来了！"

队伍瞬间乱了。
```

### 9.4 选择分支

| 行为 | 结果 |
|------|------|
| 跟着大部队跑 | 死，被骑兵追上 |
| 破庙躲藏 | 遇到朱元璋 |
| 小路逃生 | 独自活下来，偶遇朱元璋 |
| 路上投降 | 被骑兵砍死 |
| 山坡投降 | 被俘虏 |

### 9.5 破庙场景

```
【破庙】

你钻进一座破庙，里面已经有人了。

三个人蹲在佛像后面，穿着破烂的僧衣，
但腰杆挺直，和其他逃难的人不一样。

为首的那个瘦高个，
虽然脸上抹着锅灰，
但眼神沉稳，像是见惯了生死。

他旁边有根手腕粗的木棍。
另外两个和尚也各有家伙，
有的拿着扁担，有的拿着柴刀。

"进来吧，别出声。"

你刚躲好，又有两个逃难的农民钻进来。

庙外马蹄声越来越近。
元兵来了。
```

### 9.6 元兵检查

```
三个元兵端着枪进来。

为首的看了看佛像后面——六个人。
三个人手里握着木棍和柴刀。
他犹豫了一下。

"你们——"

就在这时，一个农民扑通跪下，
掏出银手镯，颤抖着递给元兵。

"军爷...小的是良民，这是家里祖传的，
军爷行行好，放过小的吧..."

元兵接过手镯，掂了掂。
他回头看了看佛像后面的几个人。

六个人，加上两个农民，八个。
他们只有三个。

元兵啐了一口：
"一群叫花子，杀了也白杀。走！"

他们转身走了。

你看向那个瘦高个的和尚。
他重新蹲下，闭着眼睛，
像什么都没发生过一样。

你知道他是谁吗？
你不知道。
你只知道——他刚才在袖子里握着什么东西，
如果元兵动手，他一定会反抗。
```

## 十、结局设计

### 10.1 常规结局

```
【结局A：朱元璋路线】
→ 你是大明开国的铁匠
→ 你的徒弟遍布天下
→ 你的铁铺成了军械局

【结局B：陈友谅路线】
→ 鄱阳湖之战，你死在那艘船上
→ 或者你跑了出来，但成了没有主子的流浪铁匠

【结局C：张士诚路线】
→ 苏州城破那天，你逃了
→ 但你活了下来

【结局D：元军路线】
→ 你被拉去打造攻城器械
→ 元朝覆灭那天，你选择了跑
```

### 10.2 逆天改命（隐藏结局）

```
【贡献度99%+，陈友谅线】

你打造了最好的武器和铠甲。
你的工匠团队遍布陈友谅的军队。
你参与了鄱阳湖的战术制定。

历史说：陈友谅应该死在这场战役里。
但因为你，他活了下来。

→ 解锁隐藏剧情：陈汉政权续命

【贡献度99%+，朱元璋线】

你打造了最好的武器。
你参与了每场重要战役。
你看着朱元璋从一个起义军首领变成皇帝。

历史说：一切本该如此。
但你的贡献，让更多人活了下来。

→ 解锁隐藏结局：洪武铁匠传
```

## 十一、设计总结

### 11.1 核心问答

**Q：你想成为什么？**
A：不需要是将军，可以只是铁匠。

**Q：你想改变历史吗？**
A：不是改变历史，而是在历史中活下来。

**Q：武器是杀戮的工具吗？**
A：刀剑可以杀戮，但铁匠是在为和平制造刀剑。

### 11.2 一句话总结

> **你不是在改变历史，你是在历史中活下来。**
>
> **——但如果你足够努力，也许能改变一点点。**
